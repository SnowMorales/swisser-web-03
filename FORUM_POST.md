## ğŸ¨ **Customization Examples**

### Change Your Server Details
Edit `src/config/site.config.json`:
```json
{
  "server": {
    "name": "Your Server Name",
    "ip": "your.server.ip",
    "discord": "your-discord-link"
  }
}
```

### Modify Color Scheme
Update `tailwind.config.js` for instant theme changes:
```javascript
colors: {
  primary: '#your-color',
  accent: '#your-accent'
}
```

### Add Your Images
Simply replace files in `/public/images/`:
- Hero backgrounds
- Gallery screenshots  
- Team avatars
- Feature graphics
